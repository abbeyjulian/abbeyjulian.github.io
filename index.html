Research

Building a GAN for Systems Workloads

Fall 2019 - Present

with Professor Avani Wildani and Vishwanath Seshagiri

Being able to predict the type of workloads a system might encounter can help design more effective and efficient systems protocols. However, workload datasets are incredibly private (because of the proprietary information it might disclose), and the few publically availble datasets are not enough to lead to good predictive models. We hope to build a Generative Adversarial Network that can generate synthetic but informative workloads that could be used in workload prediction models.

How Introductory Computer Science Students Develop Programming Style

Fall 2019 - Present

with Professor Davide Fossati, Professor Navid Hashemi, and Malvern Madondo

Emory University's introductory Computer Science course uses a system called QTest where students are able to type and execute code from within the system. We are interested in analyzing the data collected by QTest, including the temporal data of how a student created and edited a piece of code, to learn more about how students new to Computer Science develop coding strategies and style.

Anki for First-Year Medical School Anatomy

Spring 2019

with Andrew Rees, Mark Naguib, and Bailey Pearson

Anki is an open-source flashcard platform well-suited to studying for a medical school anatomy course. As part of a Medical Education research project, I consulted on the development of a Python plugin for the Anki platform that would collect and analyze data on student studying that can be aggregated to provide real-time feedback to a course professor. This work is being pilot-tested at the Vanderbilt University Medical School.
Poster presented at AMA ChangeMedEd 2019.

Computational Sustainability and Place-Based Storytelling

Summer 2018

with Dr. Douglas H. Fisher, Emily Markert, and Kamala Varma

The Region Radio project is an on-going project at Vanderbilt University investigating how Artifical Intelligence can be used to created a road trip storyteller that curates a playlist of stories, timed to the trip, that will tell stories about a place as you drive past that place. While working on this project, I looked at how to use Natural Language Processing to introduce and provide context for stories, as well as how to use techniques from Information Retrieval to characterize and measure relevance between a physical place and a text.
Late Breaking Paper in Proceedings of the 10th International Conference on Computational Creativity (pages 336-340)

Election Control Algorithmic Complexity

2017 - 2018

with Dr. Yevgeniy Vorobeychik

Given a set of voting districts and a maximum number of votes to change at each district, we proved that it is NP-Hard to determine at which districts to change votes in order to change the outcome of an election. This is consistent with similar results in other situations of election control algorithmitc study.
